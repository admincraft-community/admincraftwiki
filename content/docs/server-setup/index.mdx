# Introduction

Setting up a Minecraft server involves choosing the right software and hardware for your needs. Whether you're looking to play with friends or build a larger community, understanding the basics of server setup is essential.

<Callout type="info" title="Overview">
  Server setup involves choosing your hosting method (self-hosted or hosting service), selecting appropriate server software, and ensuring you have the right resources to run it smoothly.
</Callout>
